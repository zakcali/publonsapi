# publonsapi
un-official publons-api at:
```
https://publons.com/wos-op/api/publication/
```
reads some of data from some web of science (WOS) articles.
attention: cannot fetch data from all WOS numbers

fields you can get are:

```
wosid
doi
ArticleTitle
journalName
journalShortName
Year
authors
AbstractText

```
